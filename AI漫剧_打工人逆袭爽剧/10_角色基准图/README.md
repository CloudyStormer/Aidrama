# 角色基准图使用说明
本文件夹为全剧6个角色的多角度基准图，用于图生图锁人物长相，保证全剧人物一致，不崩脸不穿模。
## 文件夹结构
每个角色一个文件夹，内含3张不同角度的头肩基准图：
- 01_林辰：主角，黑框眼镜蓝格子衬衫
- 02_张彪：反派领导，油头西装
- 03_苏晓：女主，齐刘海卫衣
- 04_王浩：老员工，鸭舌帽T恤
- 05_李姐：行政大姐，盘发衬衫
- 06_陈总：老板，短发西装
## 使用方法
1. 生成对应人物的图片/视频时，选择「图生图」模式，根据镜头角度选择对应角度的基准图（正脸镜头用正脸基准图，侧脸镜头用3/4侧基准图）
2. 图片相似度设置为0.7-0.8，既保证人物长相一致，又能生成不同动作/表情
3. 部分角色脖子/胳膊上的小纹身是AI生成瑕疵，图生图时相似度0.7会自动覆盖，不影响锁脸效果
## 避坑提示
- 不要用纯文生图生成人物，90%会崩脸
- 不要把相似度拉到1.0，会完全复制基准图，无法生成新动作
- 不同角度的镜头用对应角度的基准图，人物更自然，不会歪脸
