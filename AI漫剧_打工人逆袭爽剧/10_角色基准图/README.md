# 角色基准图使用说明
本文件夹为全剧人物基准图，用于图生图锁人物长相，保证全剧人物一致。
## 使用方法
1. 生成对应人物的图片/视频时，选择「图生图」模式，上传对应人物的基准图
2. 图片相似度设置为0.7-0.8，既保证人物长相一致，又能生成不同动作/表情
3. 苏晓和王浩基准图上的纹身是AI生成bug，图生图时相似度0.7会自动覆盖，不影响使用
