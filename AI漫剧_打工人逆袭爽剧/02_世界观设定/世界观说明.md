# 世界观与系统设定
## 背景设定
普通互联网公司「天讯科技」，和所有中小互联网公司一样：996是福报、无偿加班是常态、领导天天画饼、抢功甩锅是日常、PUA是管理手段，所有员工敢怒不敢言，主角林辰就是其中一个被PUA了3年的软柿子产品经理。
## 核心设定：职场反PUA系统
### 触发条件
当宿主遭遇职场PUA时（包括但不限于：道德绑架、无偿加班要求、抢功甩锅、画饼、人身攻击、不合理KPI）自动激活，只有宿主能看到系统面板。
### 任务规则
系统发布的任务必须完成，完成有奖励，失败有惩罚：
| 任务类型 | 奖励内容 |
|---|---|
| 拒绝不合理加班要求 | 对应时长3倍工资+带薪休假1天 |
| 当场怼回PUA话术 | 怼得越狠奖励越高：现金红包/反PUA话术/摸鱼技能/劳动法精通buff |
| 收集1句PUA名言 | 解锁1句自动回怼金句，收集满10句解锁「自动怼人」模式 |
| 要回被拖欠的工资/加班费/赔偿 | 奖励N倍现金+劳动仲裁精通buff |
| 带动同事一起反PUA | 奖励团队buff：全员到点下班不被找事 |
| 手撕抢功/甩锅的同事/领导 | 奖励职位保护buff：3个月内不会被穿小鞋 |
### 惩罚规则
如果宿主被PUA成功（包括但不限于：自愿无偿加班、主动背锅、接受画饼、不敢反驳不合理要求）：
1. 扣除对应奖励
2. 触发倒霉buff：比如咖啡洒键盘、迟到被抓、外卖被偷、改需求改到凌晨、bug改不完
### 系统限制
- 不能让宿主直接暴富、当CEO，所有奖励都是职场相关的实际好处
- 不能违法，所有操作必须在劳动法范围内
- 只能用于反PUA，不能用来主动欺负别人
## 核心爽点逻辑
所有爽点都是打工人现实中想做但不敢做的事：
- 到点直接拎包下班，领导喊我我装没听见
- 领导让我加班我直接要3倍加班费，不给就不干
- 领导画饼我直接怼"饼太大我消化不了，直接涨工资"
- 抢我功劳我直接在大群里甩证据
- 让我背锅我直接把聊天记录甩给大领导
- 违法辞退我直接要2N赔偿，不给就仲裁
