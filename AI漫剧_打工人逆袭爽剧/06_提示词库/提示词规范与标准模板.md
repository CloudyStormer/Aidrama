# AI漫剧提示词规范与标准模板
## 一、核心原则：锁一致性
所有提示词必须严格按照公式拼接，保证全剧画风、人物、场景100%统一，避免AI生成崩坏。
## 二、标准提示词公式
```
【通用风格前缀】+【角色固定提示词】+【场景固定提示词】+【当前镜头描述（景别+动作+表情+光线）】+【负向提示词】
```
## 三、通用固定前缀（所有图片必须加在最前面）
```
AI漫剧风格，现代都市二次元，赛博朋克冷色调，线条清晰锐利，平涂+薄阴影，高对比度，4K分辨率，高细节，
```
## 四、负向提示词（所有图片必须加在最后面）

## 九、生成参数规范
- 比例：抖音/快手用9:16（1080×1920），B站用16:9（1920×1080）
- 视频时长：所有片段固定15s，和工具限制对齐
- 图生视频运动幅度：小幅度运动（人物表情变化、镜头缓慢推拉），避免大幅度动作导致人物崩坏
